irdmtools user manual

Tools

  • rdmutil work with Invenio RDM metadata from the command line
  • eprint2rdm retrieve an EPrint record via the EPrint REST API and return a simplified JSON record almost ready for important into Invenio RDM
  • doi2rdm retrieve a DOI from CrossRef and render as a simplified JSON record almost ready for import into Invenio RDM
  • people2vocabulary transfor a JSON array of Person objects into a YAML vocabularly file suitable for RDM.
